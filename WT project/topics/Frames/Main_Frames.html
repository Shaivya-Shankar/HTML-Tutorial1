HTML Frames


  1. Iframe :

    2. The <iframe> tag defines a rectangular region within the document in which the browser can display a separate document, including scrollbars and borders. An inline frame is used to embed another document within the current HTML document.

    Example

    Iframe Syntax

    An HTML iframe is defined with the <iframe> tag:

    <iframe src = "URL"></iframe>

    The "src" attribute specifies the URL (web address) of the inline frame page.

    Iframe - Set Height and Width

    Use the height and width attributes to specify the size of the iframe. The attribute values are specified in pixels by default, but they can also be in percent (like "80%").

    Example

    Code :

    <iframe src = "files/hw.html" width = "600" height="300"></iframe>

    OR
    
<iframe src="files/hw.html" style="height:300px;width:600px;"></iframe>

    Iframe - Remove the Border

    By default, an iframe has a border around it. To remove the border, add the "style" attribute and use the CSS "border" property:

    Example

    Code :

    <iframe src="bo.html" style="border:none;"></iframe>

    Iframe - Adding the Border

    With CSS, you can also change the size, style and color of the iframe's border:

    Code :

    <iframe src="ab.html" style="border:2px solid red;"></iframe>

    Iframe - Target for a Link

    An iframe can be used as the target frame for a link. The "target" attribute of the link must refer to the "name" attribute of the iframe:

    Example

    Click Me !!!

    Code :

    <iframe src="files/target.html" name="iframe_a"></iframe>
<p><a href="files/clicked.html" target="iframe_a">Click Me !!!</a></p>

    End of Iframe

  2. Framset :

    3. HTML frames are used to divide your browser window into multiple sections where each section can load a separate HTML document. A collection of frames in the browser window is known as a frameset. The window is divided into frames in a similar way the tables are organized: into rows and columns.

    Example

    Code :

    <frameset cols="30%,*,30%">
	<frame src = "a.html">
	<frame src = "b.html">
	<frame src = "c.html">
</frameset>

    This Window is Resizable...

    Frameset - Noresize Property

    The "noresize" Property is used to block the Frame resizing by the User.

    Example

    Code :

    <frameset cols="30%,*,30%">
	<frame src = "a.html" noresize>
	<frame src = "b.html" noresize>
	<frame src = "c.html" noresize>
</frameset>

    This Window is Non - Resizable...

    End of Frameset
