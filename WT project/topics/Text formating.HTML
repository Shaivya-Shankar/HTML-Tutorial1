TEXT FORMATTING

BOLD
We can make the text bold using the b tag. The tag uses both opening and closing tag. The text that needs to be made bold must be within tag. We can also use the tag to make the text strong, with added semantic importance. It also opens with and ends with tag. EXAMPLE:

THIS IS MY FIRST HTML PROGRAMME.

MY NAME IS RAHUL.

ITALICS
The I tag is used to italicise the text. It opens with I and ends with /I tag. The em tag is used to emphasize the text, with added semantic importance. It opens with em and ends with /em tag. EXAMPLE:

THIS IS MY FIRST HTML PROGRAMME.

MY NAME IS RAHUL.

MAKING OF SUBSCRIPT OR SUPERSCRIPT
The sup element is used to superscript a text and sub element is used to subscript a text. They both have opening and a closing tag. EXAMPLE:

HELLO I AM RAHUL

HELLOI AM RAHUL

HIGHLIGHTING THE TEXT
It is also possible to highlight a text in HTML using the mark tag. It has a opening tag mark and a closing tag /mark. EXAMPLE:

HELLO I AM RAHUL

HELLO I AM RAHUL

MAKING TEXT SMALLER
The small element is used to make the text smaller. The text that needs to be displayed smaller should be written inside small and /small tag. EXAMPLE:

HELLO I AM RAHUL

HELLO I AM RAHUL

STRIKING THE TEXT
The element is used to strike through the text marking the part as deleted. It also has an opening and a closing tag. EXAMPLE:

HELLO I AM RAHUL

HELLO I AM RAHUL

INSERT THE TEXT
The ins element is used to underline a text marking the part as inserted or added. It also has an opening and a closing tag. EXAMPLE:

HELLO I AM RAHUL

HELLO I AM RAHUL
